1.3 Arithmetic properties

1.3.1 Inverse of a matrix

Theorem 1.2 If A and B are both invertible matrces, we have

\[ (AB)^{-1} = B^{-1}A^{-1} \]

\[ (A^{-1})^{-1} = A \]

\[ (A^T)^{-1} = (A^{-1})^T \]
